# LifeTV UI 设计文档

## 1. 概述

本文档描述了 LifeTV 项目的 UI 设计规范，包括颜色方案、排版规则、组件设计和布局原则。该设计规范旨在确保整个应用的视觉一致性和用户体验的连贯性。

## 2. 设计理念

- **简洁现代**：采用简洁的设计语言，突出内容本身
- **用户友好**：直观的导航和交互设计，降低用户学习成本
- **视觉层次**：清晰的信息层级，引导用户关注重点内容
- **响应式设计**：适配不同屏幕尺寸和设备类型
- **品牌一致性**：统一的颜色、字体和组件样式

## 4. 排版规则

### 4.1 字体

- **主要字体**：系统默认无衬线字体（San Francisco、Roboto、Segoe UI）
- **字体风格**：简洁、易读
- **字体权重**：
  - 常规：400
  - 中等：500
  - 半粗：600
  - 粗体：700

### 4.2 字号

| 名称 | 字号 | 用途 |
|------|------|------|
| 超大标题 | 2.5rem (40px) | 应用标题、主要页面标题 |
| 大标题 | 1.8rem (28.8px) | 模块标题、二级页面标题 |
| 标题 | 1.5rem (24px) | 组件标题、卡片标题 |
| 副标题 | 1.25rem (20px) | 二级标题、列表标题 |
| 正文 | 1rem (16px) | 主要文本、内容 |
| 辅助文本 | 0.875rem (14px) | 说明文字、次要信息 |
| 小字 | 0.75rem (12px) | 标签、提示文字 |

### 4.3 行高

- 正文：1.6
- 标题：1.3
- 按钮文本：1.5

## 5. 间距规范

| 名称 | 尺寸 | 用途 |
|------|------|------|
| 极小间距 | 0.25rem (4px) | 元素内部细微间距 |
| 小间距 | 0.5rem (8px) | 元素之间小间距 |
| 中等间距 | 1rem (16px) | 常用间距、组件内边距 |
| 大间距 | 1.5rem (24px) | 模块间距、卡片间距 |
| 超大间距 | 2rem (32px) | 页面区块间距 |

## 6. 圆角规范

| 名称 | 尺寸 | 用途 |
|------|------|------|
| 小圆角 | 6px | 按钮、输入框、小卡片 |
| 中圆角 | 8px | 卡片、对话框、面板 |
| 大圆角 | 12px | 模块容器、弹窗 |

## 7. 阴影规范

| 名称 | 样式 | 用途 |
|------|------|------|
| 小阴影 | 0 2px 8px rgba(0, 0, 0, 0.08) | 卡片、按钮默认状态 |
| 中阴影 | 0 4px 15px rgba(0, 0, 0, 0.12) | 卡片悬停、弹窗 |
| 大阴影 | 0 6px 20px rgba(0, 0, 0, 0.16) | 模态框、下拉菜单 |

## 9. 布局原则

### 9.1 响应式布局

- 桌面端：侧边导航 + 主内容区
- 平板端：可折叠侧边导航
- 移动端：底部导航栏

### 9.2 网格系统

- 采用 12 列网格系统
- 最小内边距：16px
- 最大内容宽度：1200px

### 9.3 页面结构

```
┌─────────────────────────────────────────────────┐
│                   顶部导航栏                     │
├───────────────┬─────────────────────────────────┤
│               │                                 │
│   侧边导航    │       主内容区域                 │
│               │                                 │
│               │                                 │
│               │                                 │
│               │                                 │
└───────────────┴─────────────────────────────────┘
```

## 10. 图标规范

- 使用 Material Design 图标库
- 图标大小：24px（默认）
- 图标颜色：与文本颜色保持一致
- 激活状态图标：主色 (#667eea)

## 11. 交互设计

### 11.1 悬停效果

- 按钮：轻微上浮，阴影加深
- 卡片：轻微上浮，阴影加深
- 链接：颜色变化，下划线（可选）

### 11.2 点击反馈

- 按钮：轻微缩放效果
- 可点击元素：视觉反馈（颜色变化、阴影变化）

### 11.3 过渡动画

- 页面切换：淡入淡出效果
- 组件显示/隐藏：平滑过渡
- 状态变化：平滑过渡

## 12. 页面设计

### 12.1 首页

- 顶部导航栏：Logo、分类导航、搜索框
- 侧边导航：主要功能入口

## 14. 开发指南

### 14.1 组件实现

- 优先使用 Flutter 内置组件
- 自定义组件遵循设计规范
- 组件命名清晰，功能单一
- 组件参数化，支持灵活配置

### 14.2 样式管理

- 使用主题系统管理全局样式
- 组件内部样式使用常量定义
- 避免硬编码颜色和尺寸值
- 使用相对单位，确保响应式设计

### 14.3 代码规范

- 遵循 Dart 代码风格指南
- 组件代码结构清晰
- 添加必要的注释
- 保持代码简洁高效

## 15. 维护与更新
